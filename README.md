# 🩸 Blood Assist AI

Blood Assist AI is a full-stack AI-powered emergency medical assistance platform designed to connect patients with nearby blood donors in real time. The system reduces response time during critical medical emergencies using artificial intelligence, live location tracking, and instant notifications.

---

## 🚨 Problem Statement

In emergency situations, finding compatible blood donors quickly is difficult. Traditional systems depend on manual coordination, phone calls, or social media appeals, which lack real-time communication and intelligent matching.

Blood Assist AI addresses this issue by automating emergency blood requests and instantly notifying eligible donors within a specific geographic radius.

---

## ✨ Key Features

* 🎙️ AI Voice-Based Emergency Requests
* ⚡ Real-Time Donor Notification using WebSockets
* 📍 Live Location Tracking & Distance Mapping
* 🧠 Smart Donor Matching Algorithm
* 🔐 Secure Authentication using JWT
* 🏥 Blood Bank Integration
* 🌐 Multi-language Voice Support

## 🏗️ System Architecture

Blood Assist AI follows a client–server architecture integrating:

* React Frontend Interface
* Node.js Backend API
* AI Voice Processing Layer
* Real-Time WebSocket Communication
* Lightweight JSON-Based Data Storage
